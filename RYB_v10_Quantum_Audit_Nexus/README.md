# RYB v10.0 — Quantum Audit Nexus

**Release Date:** 2026-05-23  
**Codename:** Quantum Nexus  
**Status:** Production Ready

## Panoramica
RYB (Are You Broke?) è un sistema di revisione buste paga multi-agente, 
AI-native, quantum-resistant, federato e collaborativo. 
Dalla v0.0 "Genesis" alla v10.0, 55 innovazioni implementate.

## Avvio Rapido
```bash
# 1. Avvia infrastruttura
docker-compose up -d

# 2. Backend
cd backend && npm install && npm run dev

# 3. Frontend
cd frontend && npm install && npm run dev
```

## Stack
- Frontend: React 18 + Vite + TypeScript + Zustand + Tailwind
- Backend: Node.js + Express + TypeScript + BullMQ + PostgreSQL + Redis
- AI: WebAssembly OCR + Transformers.js + Federated Learning
- Security: CRYSTALS-Kyber + WebAuthn + Zero-Knowledge
- Blockchain: Notarization report su chain locale
